# newCHR_Milestones

A lightweight JavaScript library to dynamically render child milestone forms by age (months) using Tailwind CSS.

## Features

- Define milestones by month and category.
- Dynamically render Yes/No selects for each milestone.
- Tailwind CSS styled for categories.
- Can be integrated into any existing form.
